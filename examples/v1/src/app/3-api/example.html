In this example we manipulate the SVG Nodes component externally using the following exposed API methods.

method params description
addRow label Adds a new row with one node and the given label.
setNodeLabel row_index, col_index, label Sets the label for the targeted node
setNodeHighlight row_index, col_index, label Add/removes the class 'highlight' to the target node
