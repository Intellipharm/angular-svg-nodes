In this example we provide callbacks for the following events.

callback params description
on-node-selection row_index, col_index Called whenever a node is selected.
on-node-connection-change source_row_index, source_col_index, target_row_index, target_col_index, is_connected Called when a connection between two nodes is added or removed.
on-node-added row_index, col_index Called when a new node is added.
